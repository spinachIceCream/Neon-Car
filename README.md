# Neon Racer

A high-speed 2D racing game built with vanilla HTML, CSS, and JavaScript.

## How to Play
1. Open `index.html` in your web browser.
2. Click **START ENGINE**.
3. Use **Left/Right Arrow Keys** or **A/D** to switch lanes.
4. Avoid the neon traffic.
5. Survive as long as possible to increase your score.

## Features
- Smooth controls
- Neon aesthetics
- Increasing difficulty
- High score tracking (per session)
